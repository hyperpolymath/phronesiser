== What Is This?

Phronesiser adds **provably safe ethical constraints** to AI agent decision-making —
combining symbolic logic proofs with neural network adaptability.

Phronesis (Greek: practical wisdom) is a neuro-symbolic language for ethical
reasoning on the BEAM VM. Phronesiser wraps existing AI agents in ethical guardrails.

== How It Works

Define ethical constraints in `phronesiser.toml`. Phronesiser:

1. Generates Phronesis ethical reasoning modules for your AI agents
2. Symbolic logic proves safety properties ("never recommend action X")
3. Neural components handle nuanced judgment calls
4. BEAM supervision trees provide fault-tolerant execution
5. Every decision is auditable with formal justification

== Key Value

* **Provably safe AI decisions** — not just guidelines, formal proofs
* **Ethical guardrails** that are mathematically enforced
* **Neurosymbolic** — symbolic certainty + neural flexibility
* **Fault-tolerant** — BEAM VM supervision means ethics never crash
